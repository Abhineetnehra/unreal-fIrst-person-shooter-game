# ShooterGame

A fast-paced **First Person Shooter (FPS)** developed using Unreal Engine.  
The game demonstrates core FPS mechanics including player movement, weapon systems, enemy interaction, and immersive 3D environments.

# About the Game

This project is a prototype first-person shooter built to explore gameplay mechanics and level interaction using Unreal Engine.
Players navigate through a 3D environment, engage enemies using firearms, and interact with the game world through a first-person perspective.

The project focuses on core FPS systems such as:
- Player movement and camera control
- Weapon firing mechanics
- Enemy interactions
- Environmental navigation

# Features
- First-person camera system
- Weapon firing mechanics
- Real-time combat interaction
- Immersive 3D environments
- Enemy targets
- Basic damage system
- Level exploration mechanics

# Tech Stack
- Unreal Engine
- C++
- Unreal Blueprints
- 3D Assets
- Game Physics System
